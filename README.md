### 📊 影響健康支出的因素
#### 📌 專案簡介
根據 WDI 資料庫中，我們探討 2015 年影響健康支出的資料，其中以人均 GDP 和醫師密度兩個因素對健康支出呈現顯著，顯示人均 GDP、醫師密度對健康支出的影響。

透過R語言進行：從原始資料的匯入、資料清洗與缺失值處理、多變量線性回歸模型的建立，到最後使用ggplot2進行資料視覺化與趨勢分析。

#### 📊 核心發現
* **線性回歸模型**
專案中建立了多變量回歸模型來預測各國的人均醫療支出。根據分析結果，模型具備高度的解釋能力：
  * 殘差標準誤 : 591.1 (自由度 113)。
  * 決定係數 : 0.8934 / **調整後R平方向** : 0.8906，代表模型可解釋高達約 89% 的資料變異。
  * F 檢定統計量 : 315.8，且 **p-value** $< 2.2 \times 10^{-16}$，模型整體具備極為顯著的統計學意義。

* **資料視覺化**
利用ggplot2繪製了 **人均醫療支出 vs. 人口依賴比率** 的散佈圖。
* **技術工具**：R語言/R markdown(生成整合式 HTML 報告)。

### 📊 跨國人口結構與健康支出關聯性研究 (以日、韓兩國為例)
#### 📌 專案簡介
期末分析成果，主要探討 **南韓 (Korea, Rep.)** 與 **日本 (Japan)** 在 2010 年至 2023 年間，高齡人口比例、年齡依賴比與人均健康支出之間的關聯性。

利用 **R 語言** 進行資料清洗、統計分析，並透過線性迴歸模型探討人口老化對國家醫療衛生財務負擔的潛在影響。

#### 📊 核心發現
* **人口高齡化趨勢：** 資料顯示日、韓兩國的老齡人口比例在過去十多年間皆呈現顯著上升趨勢。
* **迴歸分析統計：** 專案中建立的線性迴歸模型成功揭示了「年齡依賴比」與「人均健康支出」之間的顯著關聯性。
  * 模型的 **多重R平方達 0.447**，意即年齡依賴比能解釋大約 **44.7%** 的人均健康支出變異。
  * 模型的殘差標準誤差為 1263（自由度 194），顯示預估模型仍有進一步優化與加入其他控制變數的空間。
